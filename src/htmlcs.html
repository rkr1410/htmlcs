<!--DOCTYPE html>

<!DOCTYPE html> is a required preamble at the top of an HTML document. This prevents the browser from going into quirks mode - a page redndering flow which emulates nonstandard behavior in old browsers.

Although the specification says it's case-insensitive, it's a convention. W3 Polyglot Markup requires the 'DOCTYPE' string to be uppercase, and suggests 'html' to be lowercase.
