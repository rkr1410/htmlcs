html <link> element

Describes a relation between the current document and an external resource. Typically this is a stylesheet, but can be a google font, favicon or main screen application icon for a mobile app. So, in head section:

<link href="https://fonts.googleapis.com/css2?family=..." rel="stylesheet">
In this specific case (getting a google font) we could just go to CSS and @import url('font.url');
